∫ Calculus Problems

Problem 1: Basic Derivative

Find the derivative of f(x) with respect to x:

f(x) = 3x² + 2x - 5
Use the power rule: d/dx(x^n) = nx^(n-1). The derivative of a constant is 0.

Problem 2: Chain Rule

Find the derivative of g(x) with respect to x:

g(x) = (2x + 1)³
Use the chain rule: d/dx[f(g(x))] = f'(g(x)) × g'(x). Here f(u) = u³ and g(x) = 2x + 1.

Problem 3: Basic Integral

Find the indefinite integral:

∫ (4x³ - 6x + 2) dx
Use the power rule for integration: ∫ x^n dx = x^(n+1)/(n+1) + C. Don't forget the constant C!

Problem 4: Definite Integral

Evaluate the definite integral:

∫₀² (x² + 1) dx
First find ∫(x² + 1)dx = x³/3 + x + C, then evaluate from 0 to 2: [2³/3 + 2] - [0³/3 + 0]
