♣ Expectation and Linearity

Problem 1: Basic Expected Value

A fair six-sided die is rolled. What is the expected value?

The possible outcomes are: 1, 2, 3, 4, 5, 6
Each outcome has probability 1/6
E[X] = Σ x·P(X=x) = (1·1/6) + (2·1/6) + ... + (6·1/6)

Problem 2: Linearity of Expectation

Two independent dice are rolled. What is the expected value of their sum?

Let X = first die, Y = second die
We want E[X + Y]
By linearity: E[X + Y] = E[X] + E[Y]. Each die has E = 3.5

Problem 3: Variance Calculation

For a single fair die, what is the variance? Use Var(X) = E[X²] - (E[X])²

E[X] = 3.5 (from Problem 1)
Need to calculate E[X²] = (1²·1/6) + (2²·1/6) + ... + (6²·1/6)
E[X²] = (1+4+9+16+25+36)/6 = 91/6 ≈ 15.17. Then Var(X) = 15.17 - 3.5² ≈ 2.92

Problem 4: Expected Value of a Function

A coin is flipped 3 times. Let X = number of heads. What is E[2X + 1]?

Possible values of X: 0, 1, 2, 3
P(X=0) = 1/8, P(X=1) = 3/8, P(X=2) = 3/8, P(X=3) = 1/8
First find E[X] = 0·(1/8) + 1·(3/8) + 2·(3/8) + 3·(1/8) = 1.5. Then E[2X + 1] = 2·E[X] + 1 = 4

Problem 5: Geometric Distribution Expected Value

A biased coin has probability p = 0.3 of heads. What is the expected number of flips until the first head appears?

This follows a geometric distribution
For geometric distribution: E[X] = 1/p
For geometric distribution with success probability p = 0.3, E[X] = 1/p = 1/0.3 ≈ 3.33
