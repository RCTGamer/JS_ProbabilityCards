♠ Random Walks and Martingales

Problem 1: Simple Random Walk

A particle starts at position 0. Each step, it moves +1 or -1 with equal probability. What's the probability it reaches +3 before reaching -2?

+3 ← Target
+2
+1
●→→ 0 ← Start
-1
-2 ← Barrier
This is a gambler's ruin problem
Starting at 0, target at +3, barrier at -2
Relative to barrier: start = 2, target = 5, barrier = 0
For symmetric random walk, P = (starting distance from barrier) / (total distance) = 2/5 = 0.4

Problem 3: Martingale Property

Let S_n be a simple random walk. Is X_n = S_n² - n a martingale?

S_n = S₀ + ξ₁ + ξ₂ + ... + ξₙ where ξᵢ = ±1 with prob 1/2
X_n = S_n² - n
Check if E[X_{n+1} | S₀, S₁, ..., S_n] = X_n
E[S_{n+1}² | S_n] = E[(S_n ± 1)²] = S_n² + 1. So E[X_{n+1} | S_n] = S_n² + 1 - (n+1) = S_n² - n = X_n. Yes, it's a martingale!

Problem 4: Stopping Time

A fair coin is flipped repeatedly. Let T = min{n : we see HH or TT}. What is E[T]?

T is the first time we see two consecutive identical outcomes
This is a stopping time problem
We can solve using states: ∅, H, T, HH/TT
Set up equations: E[T] = 1 + E[T|first flip]. By symmetry, E[T|H] = E[T|T]. From state H: 1/2 chance done (T=2), 1/2 chance restart. Solving: E[T] = 3.

Problem 5: Optional Stopping Theorem

For simple random walk starting at 0, let τ = min{n : S_n = 5 or S_n = -3}. What is E[S_τ]?

S_n is simple symmetric random walk
τ is stopping time when hitting +5 or -3
S_n is a martingale, and τ has finite expectation
By optional stopping theorem, E[S_τ] = E[S_0] = 0, since S_n is a martingale and τ is a bounded stopping time.

Problem 6: Doob's Martingale

If X_n = 2^{S_n}/2^n where S_n is simple random walk, what is E[X_n]?

S_n = S₀ + ξ₁ + ... + ξₙ where ξᵢ = ±1 with prob 1/2
X_n = 2^{S_n}/2^n
This is an example of an exponential martingale
E[2^{ξᵢ}] = (1/2)·2¹ + (1/2)·2⁻¹ = 1 + 0.5 = 1.5. But E[X_n] = E[2^{S_n}]/2^n = (E[2^{ξ₁}])^n/2^n = (1.5)^n/2^n... Wait, recalculate: E[2^{ξᵢ}] = 1.5, but the martingale property gives E[X_n] = X_0 = 1.
