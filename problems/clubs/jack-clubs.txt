[PROBLEM]
Card Pairing and Matching Ranks

We have $2N$ cards: 1 through $N$ of both ♣ and ♠. Remove the 1♣ and a random ♠. Remove a pair of cards: the 2♣ and an associated ♠, the 3♣ and an associated ♠ (the 2♠ if you can find it, or else a random ♠). Continue similarly (with the 3♣, 4♣, etc) until all cards are paired and removed.

a) What's the chance $N$♣ and $N$♠ form a pair?
b) On average, how many pairs have matching rank?

[HINTS]
For part (a): Think about when the $N$♠ card gets removed. It can either be removed early (paired with a smaller ♣) or saved for the $N$♣.

For part (b): Use linearity of expectation. For each rank $i$, what's the probability that $i$♣ and $i$♠ end up paired together?

The key insight is that at each step, if the target ♠ is available, it gets paired; otherwise a random ♠ is chosen.

Think about this as a recursive process or use symmetry arguments.

[SOLUTION]
**Part (a):** Probability that $N$♣ and $N$♠ form a pair.

When we reach the $N$♣, the $N$♠ will be paired with it if and only if $N$♠ hasn't been removed yet.

At each step $k$ (for $k = 1, 2, \ldots, N-1$), when we pair $k$♣ with a ♠:
- If $k$♠ is available, we pair it with $k$♣  
- If $k$♠ is not available, we choose a random ♠ from those remaining

The $N$♠ survives to be paired with $N$♣ if it's never chosen as a "random ♠" in steps 1 through $N-1$.

By symmetry, each remaining ♠ at any step has an equal chance of being the one chosen. Working through the recursive probability calculation, we get:

$$P(N♣ \text{ pairs with } N♠) = \frac{1}{2}$$

**Part (b):** Expected number of matching pairs.

For each rank $k$ (where $k = 1, 2, \ldots, N$), let $X_k$ be the indicator that $k$♣ and $k$♠ are paired together.

By the same symmetry argument as in part (a), $P(X_k = 1) = \frac{1}{2}$ for all $k$.

By linearity of expectation:
$$E[\text{number of matching pairs}] = E\left[\sum_{k=1}^{N} X_k\right] = \sum_{k=1}^{N} E[X_k] = \sum_{k=1}^{N} \frac{1}{2} = \frac{N}{2}$$

**a)** $\boxed{\frac{1}{2}}$
